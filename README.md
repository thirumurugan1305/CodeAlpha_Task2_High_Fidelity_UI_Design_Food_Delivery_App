# CodeAlpha UI/UX Design Internship

## Task 2 – High-Fidelity UI Design

### 📌 Project
Food Delivery App

### 🛠 Tool Used
- Figma Make (AI-assisted)
- Figma

### 📖 Description
This project was completed as part of my UI/UX Design Internship at CodeAlpha.

The objective was to convert a low-fidelity wireframe into a high-fidelity user interface by applying UI/UX principles, modern layouts, typography, color palettes, icons, and interactive design elements.

### 📱 Features
- Login & Authentication
- Home Screen
- Restaurant Listings
- Shopping Cart
- Checkout
- Order Confirmation

### 🎯 UI/UX Principles Applied
- Consistent color palette
- Visual hierarchy
- Responsive layout
- User-friendly navigation
- Clean typography
- Component consistency
